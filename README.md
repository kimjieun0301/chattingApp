# Chatting App
실시간 채팅 프로그램


## 🖥️ 프로젝트 소개
여러 메신저를 참고하여 만든 실시간 채팅 프로그램입니다.
<br>

## 🕰️ 프로젝트 기간
* 23.06.12일 - 23.07.21일

### ⚙️ 개발 환경
- `C#`
- **IDE** : Visual studio
- **Framework** : .NET framework 6.0
- **Database** : Tibero 6
